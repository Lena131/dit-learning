# git-learning

текстовый репозиторий для изучения git

## Изучили команду git clone ##

Команда git clone позволяет копировать или клонировать к себе репозитории из
интернета.
Для этого мы обращаемся к программе Git, указываем параметр clone и
после этого даём адрес того репозитория, который надо скачать.
Далее в папке, где вызывается команда git clone, появляется новая папка. 
И уже внутри этой папки будет лежать полная копия указанного нами репозитория
